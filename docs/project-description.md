# Saturn X Project Description

Smart task planner that helps students divide their school work in an efficient and simple way.


# Features:

* Users can create an account or login using Google or GitHub (May change to only Google
for calendar sync purposes)

* Users can customize their study/work habits by filling out a form
that they will get when they register. i.e., customer can save in their planner 
settings that they want to do hardest tasks first.

* Users can submit a file or write an assignment/task description to add a new assignment
to their planner

* Users must specify the due date for the task

* When users enter the due date they can see how many days remaining

* Users can then create a new task to be planned and will see an AI generated
plan agenda with sub-tasks to complete each day. 

* Users can modify the AI generated tasks and change the date, time, or task description

* Users can add additional tasks to complete 

* Users can see all their tasks in a calendar view that can be changed to weekly or monthly
view.

* Users can mark a task as complete. 

* Users will see a streak of n days if they have completed all the tasks for the past n days.

* User's tasks can be synced with their Google Calendar. 

* User can get daily reminders of all the tasks that they have to complete each day.
